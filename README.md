autsmuxer
=====

* mkv2vob CLI clone for *nix.

## dependencies:

* [libdca][libdca]
* [mkvtoolnix][mkvtoolnix]
* [mencoder][mencoder]
* [cmake][cmake]
* [aften][aften]
* [tsMuxeR][tsmuxer]
* [spdifconvert][spdifconvert]

## usage:

* Install all dependencies, ensuring they are in $PATH.
* bash autsmuxer.sh [-options] inputfile/dir.

## notes:

* mkvtoolnix must be >5.4.0.
